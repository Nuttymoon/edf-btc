## ðŸ“¤ Generated Data (from Scripts)

These files are generated by running the scripts in `/scripts/`. See [scripts/README.md](/scripts/README.md) for details.

| File | Generated by | Description |
|------|--------------|-------------|
| `nuclear-surplus.csv` | `compute-surplus.ts` | Monthly nuclear energy surplus (optimal - actual production) |
| `bitcoin-mining-monthly.csv` | `bitcoin-mining-monthly.ts` | Aggregated monthly Bitcoin network data (hash rate, BTC created, price) |
| `mining-simulation.csv` | `mining-simulation.ts` | Full simulation results (miners, hash rate, BTC mined/accumulated, CAPEX) |

---

## ðŸ”„ Regenerating Data

To regenerate the output files:

```bash
npm run compute-surplus
npm run bitcoin-mining-monthly
npm run mining-simulation
```

**Note:** Run scripts in order as `mining-simulation.ts` depends on outputs from the other two.
